A Flutter application that allows users to **scan QR codes** and **generate custom QR codes** easily.  
The app provides a simple and clean interface for creating QR codes from text/URLs and scanning QR codes using the device camera.

## Features

- Generate QR codes from text and URLs
- Scan QR codes using the camera
- View scanned QR results
- Clean and simple Flutter UI
- Splash screen
- Responsive design
